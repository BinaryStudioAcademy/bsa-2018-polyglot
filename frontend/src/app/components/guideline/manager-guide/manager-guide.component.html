1. Create a project

To create a project go here and submit a form with your data.

After that your project will appear on the dashboard.

2. Create a string

To create a string go to the project`s workspace. All information about your strings will be displayed there.

Press Add new string button in the top right corner.

Here you have to type your string`s key, base value, description, select tags and picture.

info_outline You should select all names suitable to make translator`s work easier!

3. Create a team

To create a team go here and choose translators from general list.

All translators will get an invitation from you.

info_outline Pay attention to translator`s language level!

Now you have to assign this team to your project.

Go to Teams tab in your`s project details and assign your team.

4. Create glossary

To create a glossary go here and press "+" button.

Now fill glossary with terms and explanations that relate to your technology.

After that you have to assign glossary to your project in the same way like you have assigned team.

5. Watch the progress

Now your translators are working and you can watch how the work goes.

On the dashboard you can see general progress and in Report tab you can see detailed information.
