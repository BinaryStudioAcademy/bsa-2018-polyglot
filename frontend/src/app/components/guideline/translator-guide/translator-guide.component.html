1. Get a team invitation

First of all, you should find a manager that is looking for translators.

As soon as he invites you into his team, you will get a notification.

To enter this team, you have to accept it.

2. Translating

To start your work, you have to enter the project's workspace.

In the left column you will see a list of strings.

After selecting a string, you will see a list of translations for this string in the central column.

Just click on the line with an empty translation and type there your translation.

info_outline You can't type if translations are assigned to somebody else!

3. Machine translation

To make your work easier, you can use machine translation.

After clicking on the machine translation icon, you will get a dialogue window with a suggested translation.

You have to click on it to confirm this translation.

info_outline Remember that machine translation is still not ideal!

4. Glossaries

You can use glossaries to make complicated terms clearer.

To use a glossary, it should be already assigned to the project.

All you need is to type a term out of this glossary, and defenition will appear.

5. Workflow helpers

In the right column of the workspace you can find several useful options.

In the History tab, you can see all the previous versions of a selected translation.

In the Comment tab, you can see a translator's discussion about the string.

In the Optional Translation tab, you can see the other versions of a selected translation.

info_outline You should look over these tabs from time to time!
