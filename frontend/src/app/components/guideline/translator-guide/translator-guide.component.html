1. Get team invitation

First of all you should find a manager that is looking for translators.

As soon as he invites you in his team, you will get a notification.

To enter this team you have to accept it.

2. Translating

To start your work you have to enter project`s workspace.

In the left column you will see a list of stings.

After selecting string you will see a list of translations for this string in the central column.

Just click on the line with empty translation and type there your variant.

info_outline You can`t type if translations are assigned for somebody else!

3. Machine translation

To make your work easier you can use machine translation.

After clicking on the machine translation icon you will get a dialog window with suggested translation.

You have to click on it to confirm this translation.

info_outline Remember that machine translation is still not ideal!

4. Glossaries

You can use glossaries to make complicated terms clearer.

To use the glossary it should be already assigned to the project.

All you need is to type some term out of this glossary.
